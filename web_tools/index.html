Useful Embedded Linux tools and sites

NXP IMX8MP Pad settings calculator

A simple bit calculator for determining the PAD settings value for a Linux Device tree. This is currently for the NXP IMX8MP SoC only.

Elixir Cross Referencer

An excellent symbolic search engine by Bootlin with clickable navigation to definitions and declarations for a project. Indexes C source and headers, Kconfigs, and more. Available for Linux, U-Boot, ARM Trusted Firmware, Busybox and more!

kernelconfig

Similar to Elixir, kernelconfig.io indexes all of the Kconfig options within different versions and architectures of Linux. It allows you to quickly search options, get the help text, and understand dependencies.
